# Voxel Minecraft Web (Prototipo)

Prototipo minimal de "Minecraft 3D" en Web usando Three.js.

Características incluidas:
- Mundo voxel simple (generación procedimental muy básica).
- Colocar / romper bloques (click derecho/izquierdo).
- Animales pasivos (cajas que caminan).
- Día/noche simple.
- Hotbar mínima.

Ejecutar:
1. npm install
2. npm run dev
3. Abrir http://localhost:5173
